# Choice Kitchen Landing Page

A responsive Next.js + Tailwind landing page for Choice Kitchen restaurant.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open `http://localhost:3000` in your browser.

## Project files

- `app/page.tsx` — landing page implementation
- `app/layout.tsx` — root layout and metadata
- `app/globals.css` — Tailwind imports and page styling
- `tailwind.config.ts` — Tailwind configuration
- `postcss.config.js` — PostCSS configuration
- `package.json` — project dependencies and scripts

## Notes

The workspace currently does not have `node` or `npm` available in the shell, so dependencies must be installed in an environment where Node.js is installed.
